This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

This is an MCP (Model Context Protocol) server for the Hunter HydroWise smart irrigation platform. It exposes HydroWise API functionality as MCP tools so Claude can interact with irrigation controllers, zones, schedules, and watering data.
